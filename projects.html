Carlos Pisani Data Projects

Review Hound

An application that is meant to help catch Opinion Fraud in hotel reviews. Currently, it analyses a review and creates a true "Sentiment" score based on the context. This is done via a Keras Deep Neural Network model which was trained on Tripadvisor hotel review data. THe model is served via a Flask framework on Heroku. The reviews were stored in a SQL Lite Database and queried via DB Browser. 4&5 Star reviews were given a score of 1, 1&2 a score of 0. Another model was created to determine if a review was "incentivized" based on the context. This Tensorflow Categorical model will be served to the public in the near future.

Birding Oracle

Created a comprehensive bird sighting applicaiton using a Flask framework with Plotly and Leaflet visualizations. The user is able to search for birding hotspots by state/region or can also search for specific species. Bird data is being retrieved from the Ebird API and queried via Python scripts. Bird Images were scraped from the web and stored in a SQL Lite Database. The top 10 birding locations and top 5 species per state/region visualizations are populated via Javascript.Bird enthusiats are encouraged to use this interactive tool to view recent migration pattern species or to plan their next birding trip!

Animal Shelter Analysis

A deep dive into animal shelter data for both Austin TX and Louisville KY. Pandas was the main explaratory tool to access API's from the City of Austin website and query into Data Frames. Heat maps using Google API's and Matplotlib visualizations were created to show the findings.
